# Ecommerce Website

A **full-stack Ecommerce Web application** with separate **frontend** and **backend** modules. This project demonstrates modern web development practices including CRUD operations, database integration, and modular architecture.

---

## 🛠️ Technology Stack

**Frontend:**  
- React.js  
- HTML, CSS, JavaScript  
- Axios / Fetch API for API integration   

**Backend:**  
- Java Spring Boot  
- H2 in-memory database  
- REST APIs  

**Other Tools:**  
- Git for version control  
- Postman for API testing  

---

## ⚡ Features

- View list of products  
- Add, update, and delete products (admin functionality)  
- Modular frontend and backend  
- API integration between frontend and backend
